# Welcome

## Before starting

Go to the `src/asl` folder:
```shell
$ cd src/asl
```

## Installation
```shell
$ npm install
$ typings install
```

## How to run the project
```shell
$ node dist/main.js
```
## How to run the tests
```shell
$ npm test
```
### Tip
Install **jest** as a global to watch files...
```shell
$ npm install -g jest
```
...then:
```shell
$ jest --watch
```
